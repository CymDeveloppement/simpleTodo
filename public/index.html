Aide - Utilisation de SimpleTodo
Gestion des tâches
  • Ajouter une tâche : Utilisez le champ de saisie en bas de la page, puis cliquez sur "Ajouter".
  • Marquer comme terminée : Cliquez sur le bouton rond pour marquer une tâche comme complétée.
  • Actions sur une tâche : Cliquez sur le menu (⋯) pour accéder aux actions :
    • Je m'en occupe : Assignez-vous une tâche (si elle n'est pas déjà assignée)
    • Changer de catégorie : Déplacez une tâche dans une autre catégorie
    • Modifier la date d'échéance : Ajoutez ou modifiez une date d'échéance pour une tâche
    • Supprimer : Supprimez définitivement une tâche
  • Commenter : Cliquez sur l'icône de message pour ajouter des commentaires à une tâche.
Catégories

Organisez vos tâches par catégories :

  • Dans les paramètres (), accédez à la gestion des catégories
  • Créez des catégories avec une couleur personnalisée
  • Les tâches sont automatiquement regroupées par catégorie dans des accordions
  • Ajoutez une tâche sans catégorie, puis changez-la plus tard avec le bouton tag orange
Paramètres

Cliquez sur l'icône pour accéder aux paramètres :

  • Pseudo : Personnalisez votre pseudo affiché sur vos actions
  • Thème du header : Choisissez un gradient de couleur pour le header (persistant)
  • Notifications email : Inscrivez-vous pour recevoir des alertes par email sur les nouvelles tâches et les tâches terminées. Un email de bienvenue sera envoyé avec un lien unique.
  • Liste des abonnés : Consultez qui est abonné aux notifications de cette liste, avec possibilité de renvoyer le lien
  • Inviter quelqu'un : Envoyez un email d'invitation à un collaborateur avec le lien direct vers la liste
  • Catégories : Gérez vos catégories (création, suppression)
  • Nettoyage : Supprimez toutes les tâches terminées en un clic
Notifications par email
  • Inscription : Abonnez-vous pour recevoir des notifications par email
  • Email de bienvenue : Vous recevrez un email avec un lien unique vers la liste
  • Nouvelles tâches : Notification automatique quand une tâche est ajoutée
  • Tâches terminées : Notification quand une tâche est marquée comme complétée
  • Désinscription : Annulez vos notifications via le bouton dans les paramètres
Invitation par email
  • Inviter : Dans les paramètres, section "Inviter quelqu'un"
  • Email d'invitation : Envoyez un email avec votre nom et le lien vers la liste
  • Accès direct : L'invité clique sur le lien et accède immédiatement à la liste
  • Sans inscription : Aucune inscription nécessaire pour collaborer
Partage
  • Bouton Partager : Cliquez sur le bouton "Partager" en bas de la liste pour copier le lien de la liste (sans token). Partagez-le avec vos collaborateurs pour qu'ils puissent accéder à la liste.
  • Invitation par email : Utilisez la section "Inviter quelqu'un" dans les paramètres pour envoyer un email avec un lien direct vers la liste.
  • Liens de notification : Les abonnés aux notifications reçoivent des emails avec des liens tokenisés qui les reconnectent automatiquement.
Conseils
  • Utilisez les catégories pour organiser vos tâches par projet ou priorité
  • Les tâches assignées affichent le nom de la personne qui s'en occupe
  • C'est le créateur ? Vous pouvez modifier le titre, changer le thème, supprimer les tâches terminées ou même supprimer la liste.
  • Les commentaires permettent d'ajouter des détails ou de discuter sur une tâche
  • Abonnez-vous aux notifications pour rester informé des changements
  • Partagez le lien de la liste avec le bouton "Partager" en bas
  • Invitez vos collaborateurs par email pour faciliter l'accès à la liste
  • Les emails incluent des liens directs vers la liste pour un accès facile
  • Les dates d'échéance permettent de prioriser vos tâches
